Introduction

Writing descriptive paragraphs can be successful as one of the first writing activities for students. Start by helping students understand the difference between simple and complex sentences, and move on to practice writing complex sentences. Students should also be familiar with a wide range of descriptive adjectives. Start by having students answer basic questions below. Next, use the writing exercise to expand the answers into a well-formed descriptive paragraph. Descriptive paragraphsare often used to describe what a person looks and acts like. Read this example descriptive paragraph, notice how descriptive paragraphs are arranged by putting together all the sentences about the same thing.
